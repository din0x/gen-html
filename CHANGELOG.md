# Changelog

## `0.4.0`

- `let` bindings in templates
- Optional attributes via `Value` trait
- Template expander v2, this will allow static templates it the future. 

## `0.3.0`

- Add `@` and `.` shorthands
- Fix `else if` parsing
- Implement framework specific traits for `Raw`, `Escaped` and `RenderFn`
